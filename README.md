# LPF
Low-pass filter included delay effect,  
can use as `USER DELAY FX`.

# Parameters
- time  
delay time.
- depth  
feedback gain.
- delay + depth  
dry/wet mixing ratio.
- param4  
LPF type (0..1)

# How to build
1. Clone (or download) this project.
1. Copy to under `platform/nts-1_mkii`.
1. See README.md under `platform/nts-1_mkii`.
